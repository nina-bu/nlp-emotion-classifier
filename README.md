# Emotion Detection in Written Sentences

**Author:** [Nina Bu](https://github.com/nina-bu)

## Project Description

The goal of this project is to detect emotions in written sentences and classify them based on the expressed emotions. Each sentence is classified into one of the following six emotions: sadness, anger, love, surprise, fear, and joy.

![poster](https://github.com/nina-bu/nlp-emotion-classifier/assets/116764953/0f7b5443-d9b0-43f0-b763-c06d550325c4)

## Dependencies

The following Python libraries are required to run this project:

- Emoji
- Joblib
- Keras
- Matplotlib
- NLTK
- NumPy
- Pandas
- Scikit-learn
- Seaborn
- TensorFlow
- Wordcloud

## Usage

1. Clone the repository to your local machine.

2. Run the notebook file `emotion-classification.ipynb` to execute the project. Make sure you have Jupyter Notebook or JupyterLab installed, or run it on [Kaggle](https://www.kaggle.com/code/nina6u/emotion-classification/notebook).

3. After running the notebook, trained models will be saved in the project directory.

4. To run the `main.py` file, ensure that the saved models are placed in the project directory.

5. Execute `main.py` using the following command:
```shell
python main.py
